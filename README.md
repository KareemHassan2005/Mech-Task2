# Batman Project 🦇

A 3D printable Batman model created with precision and attention to detail.

## Overview

This project features a high-quality Batman figurine designed for 3D printing enthusiasts. The model captures the iconic silhouette of the Dark Knight, optimized for reliable printing results.

## Features

- **Optimized for 3D Printing** – Clean geometry with proper wall thickness
- **Supports Included** – Pre-configured support structures where needed
- **High Detail** – Authentic Batman design with recognizable features
- **Print Ready** – STL file prepared for slicer software

## File Format

- `.stl` – Standard Tessellation Language, compatible with all major slicers (Cura, PrusaSlicer, Bambu Studio, etc.)

## How to Print

1. Download the STL file from this repository
2. Import into your preferred slicer software
3. Slice with your usual PLA/PETG settings
4. Print and enjoy your Batman figure!

## Recommended Settings

| Setting | Value |
|---------|-------|
| Layer Height | 0.2 mm |
| Infill | 15-20% |
| Supports | Yes (included in model) |
| Orientation | As provided |
